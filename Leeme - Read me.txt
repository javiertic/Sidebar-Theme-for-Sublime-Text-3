SUBLIME TEXT 3 Javiertic Dark Sidebar Theme (Based in Cobalt2 theme)

I recommended using with Sunburst theme. 

How to install:

1. Open Sublime Text 3 and go to Preferences -> Browse Packages.
3. Copy the Theme - Javiertic folder into the Packages folder.
4. In ST3 go to Preferences -> Setting - User and paste the following line of code:
	
	"theme": "Javiertic.sublime-theme" 


5. Restart ST3 and you're done.
